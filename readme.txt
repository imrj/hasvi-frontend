A wordpress interface to the Hasvi backend

== Description ==
Hasvi is a package that gives small data stream storage and visualisation capabilities to Wordpress. These data streams are typically used for Internet of Things (IoT) devices pushing regular updates of their sensor inputs or overall state. An example would be a temperature sensor in a house.
